# Ventas (MVP)

App web local para ventas e inventario.

## Requisitos
- Python 3.10+

## Instalacion
1. Crea un entorno virtual y activa.
2. Instala dependencias:
   ```bash
   pip install -r requirements.txt
   ```
3. Ejecuta:
   ```bash
   python app.py
   ```
